Neural Network

prob: high training error - high bias sol: consider more features -> use larger neural network

prob: high correlation error - high variance sol: cut weights of some features -> use more data

Understanding comes from talent & intelligence, Deduction comes from learning & experience

Iterative Loop of ML development

  1. Choose architecture (model, data etc)
  2. Train model
  3. Diagnostics(bias, variance, error analysis)

AI = Code + Data

What if you don’t have data?

  1. Hand labeling critical data bins
  2. Synthesis data by adding variations of existing datasets (ex. rotate text fonts)

Transfer Learning: Supervised pre-training + fine tuning

  • Only train output layers parameters
  • train all parameters

Decision tree

recursively picking the tree split that reduce enthropy the best

Random Forest algorithm: Use multiple distinctive decision trees XGBoost Tree: train the tree on samples that doesn’t preform well in previous iterations
